# Cadence setup

## Create a new project library

Navigate to `~/Cadence`. Open Virtuoso by typing `virtuoso &`. From the main Virtuoso window, navigate to **File &rarr; New &rarr; Library** to open the Library Manager. Enter the desired library name (e.g., **ER_ASIC_Rev1_Tapeout**), and select "Attach to an existing technology library." Attach the new library to the tsmcN65 technology library in the subsequent window.

To open the library, from the main Virtuoso window, navigate to **Tools &rarr; Library Manager**.

## Import GDS files to Cadence

Now, you will need to import the GDS files, including the pad-level GDS, into Cadence. From the main Virtuoso window, navigate to **File &rarr; Import &rarr; Stream**. Fill in the dialog accordingly:

- **Stream File**: `DigitalCorePads.gds2`
- **Library**: ER_ASIC_Rev1_Tapeout
- **Top Cell**: DigitalCorePads
- **Attach Tech Library**: tsmcN65
- **Layer Map &rarr; File Name**: `TSMC_65nm_PDK/tsmcN65/tsmcN65.layermap`
