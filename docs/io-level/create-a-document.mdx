# Synthesis

If you run synthesis as is, the pads will be optimized away because they have no logic connections. In `syn.tcl`, you have to do `set_dont_touch` for all of your pads:

```tcl title="syn.tcl"
   if {$vars(use_pad)} {
      set ALL_PAD [get_cells {PI_* PO_* PTRI_* PP_* *_CORNER}]
      set ALL_PIN [get_nets -of_objects [filter_collection
   [get_pins -of_objects $ALL_PAD] "name =~ *PAD*"]]
      set_dont_touch $ALL_PAD
      set_dont_touch $ALL_PIN
   }
   set_dont_touch {VDDPST* VSSPST*}
```

Before running synthesis, ensure that the `clk_list` is updated and that each clock is set to be asynchronous. The chip select timing may need to be handled as well. Look at `syn.tcl` in **DigitalCorePads** for reference.
