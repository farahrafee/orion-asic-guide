# GDS export

Once you are ready to export your design, go to the main Virtuoso window and navigate to **File &rarr; Export &rarr; Stream**.

You should be ready to submit your design! Be sure to check that you have completed these tasks before moving forward:

1. Each DRC must be passed (or waivable) and all summary files saved
2. GDS file must be exported from Cadence
3. LVS must be passed (green check mark + smiley face)
4. Bond pads and seal ring are both present
5. One I/O pad must be VDDPST (POC)
