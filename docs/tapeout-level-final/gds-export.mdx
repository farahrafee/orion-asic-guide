# GDS export

Once you are ready to export your design, go to the main Virtuoso window and navigate to **File &rarr; Export &rarr; Stream**.

Be sure to check that you have completed these tasks:

1. **REQUIRED**: Each DRC must be passed (or waivable) and all summary files saved
2. **REQUIRED**: GDS file must be exported from Cadence
3. Pass LVS (green check mark + smiley face)
4. Bond-pads and seal-ring are both present
5. Ensure that one I/O pad is VDDPST (POC)
