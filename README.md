# Backend Flow Docs

Documentation site live at: https://farahrafee.github.io/orion-asic-guide

## Setup
Clone the repo and install dependencies:
	git clone https://github.com/farahrafee/orion-asic-guide.git
	cd orion-asic-guide
	npm install

## Commands

| Command | Description |
|---|---|
| `npm start` | Preview site locally at localhost:3000 |
| `npm run clear` | Clear cache (fixes most errors) |
| `npm install` | Install dependencies (run after cloning) |
| `GIT_USER=farahrafee npm run deploy` | Deploy to live site |

## Editing
- All pages live in the `docs/` folder as `.md` or `.mdx` files
- Changes only go live after running the deploy command
